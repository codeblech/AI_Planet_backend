PDF QA Service Documentation
===========================

A FastAPI-based service that processes PDF documents and provides a question-answering interface using Gemini AI.

Features
--------
* PDF file upload with validation
* WebSocket-based real-time communication
* Integration with Google's Gemini AI
* Vector database storage using ChromaDB
* Rate limiting
* Session management
